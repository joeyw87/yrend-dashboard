# yrend-dashboard
연말정산 진행현황 대시보드

# 📊 2025 연말정산 진행현황 통합 대시보드

**2025년 귀속 연말정산 데이터의 진행 상태를 시각적으로 파악하고 관리할 수 있는 웹 기반 대시보드입니다.**

별도의 서버 구축 없이 브라우저상에서 로컬 JSON 데이터를 로드하여 즉시 분석할 수 있으며, PC와 모바일 환경 모두에 최적화되어 있습니다.

## 🚀 주요 기능 (Key Features)

* **📈 데이터 시각화**: SaaS형 고객 기준 연말정산 대상자, 제출/미제출 현황, 진행률 등을 차트와 지표로 한눈에 확인
* **📂 JSON 데이터 연동**:
    * **업로드**: 로컬에 있는 JSON 파일을 드래그하거나 선택하여 대시보드 데이터 즉시 갱신
    * **다운로드**: 현재 보고 있는 분석 데이터를 JSON 파일로 백업 및 공유 가능
* **📱 반응형 디자인 (Mobile Responsive)**: 데스크탑은 물론 스마트폰에서도 깨짐 없이 최적화된 화면 제공
* **⚡ 실시간 필터링**: '계속근무자' / '중도퇴사자' 등 조건에 따른 데이터 필터링 토글 기능 제공

## 🛠️ 사용 방법 (How to Use)

1.  **데이터 준비**: 연말정산 데이터가 담긴 JSON 파일을 준비합니다. (기본 제공되는 샘플 데이터 다운로드 가능)
2.  **파일 업로드**: 대시보드 상단의 `파일 선택` 버튼을 누르거나 `upload-area`에 파일을 드래그합니다.
3.  **현황 분석**: 자동으로 갱신된 차트와 수치를 통해 진행 상황을 모니터링합니다.
4.  **필터링**: 우측 상단의 `계속근무자만` 토글을 사용하여 원하는 대상만 추려서 확인합니다.
